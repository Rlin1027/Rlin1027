# Overview

**Full-stack engineer based in Taiwan**, building at the intersection of AI, automation, and civic technology.

I enjoy turning complex problems into clean, user-facing products — from personal AI assistants and ML research tools to gamified civic-data platforms and production automation pipelines.

**ML / Deep Learning Research** — enhanced training utilities for KAN, reinforcement learning, and GPT architectures
**Civic Tech & Finance** — gamified government data platforms, multi-model stock valuation systems
**Automation** — n8n workflows for data pipelines, music rendering, and RAG systems
**Full-Stack Apps** — real-time dashboards, playlist organizers, visual storytelling tools
